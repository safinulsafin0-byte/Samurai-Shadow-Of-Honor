# Samurai-Shadow-Of-Honor
Crimson Katana is a third-person samurai action game developed in Unreal Engine 5. Fight against deadly enemies using fast sword combat mechanics, survive intense battles, and master the way of the blade in a cinematic Japanese-inspired world.
# Crimson Katana

A third-person samurai combat game made with Unreal Engine 5.

## Features
- Sword Combat System
- Enemy AI
- Health System
- Kill Counter
- Game Over Screen
- Cinematic Samurai Theme
